CineStock Vert

Features

  • User Authentication Secure user login, registration, and role management (admin, department) using Firebase Authentication. The application stores the current user to Firestore, along with project assignments.
  • Project Management Creation, selection, and management of film projects, associating users with specific projects and roles stored in Firestore. Storing these parameters allows role-based permission management for all Firestore collections.
  • Inventory Management CRUD operations for managing inventory items, including name, quantity, unit, status (pending_order, in_stock, virtual_stock, donated), project ID, department, and user ID. Implemented via Firestore.
  • AI-Powered Inventory Upload
    AI
     Uses Google Gemini via a Firebase Cloud Function to analyze uploaded files (PDF, CSV, images) and extract a structured list of inventory items and quantities, displayed in a confirmation modal.
  • Circular Economy Management Cloud Function triggered when an inventory status is updated to virtual_stock or donated, creating a new inventory entry and reducing the original item quantity, enabling circular economy practices.
  • CSV Export Generates a CSV file of inventory data (filtered by project ID and status) via a Firebase Cloud Function and provides a download URL.
  • Responsive UI/UX Mobile-first design with a responsive sidebar, dashboard grid, and streamlined forms for easy use on various devices, leveraging the useResponsiveLayout hook to apply layout strategies according to device, prioritizing one-handed usage on mobile.
